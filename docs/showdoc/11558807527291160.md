# 根据biz获取公众号基本信息

##### 简要描述

- 根据biz获取公众号基本信息（关于公众号页面）

##### 收费标准

- `按调用次数：5分钱/次`

##### 请求URL

- ` http://data.wxrank.com/weixin/getbiz`

```
curl --location 'http://data.wxrank.com/weixin/getbiz?key=xxx&biz=MzI2NDk5NzA0Mw=='
```

##### 请求方式

- GET / POST

##### 参数

| 参数名 | 必选 | 类型   | 说明      |
| :----- | :--- | :----- | --------- |
| key    | 是   | string | 密钥      |
| biz    | 是   | string | 公众号biz |

##### 备注

> `code 状态判断`

- 0：获取成功
- 1000：积分不足
- 1001：biz不能为空
- 9999：QPS超过上限，请稍后再试（每秒不能超过3个）

> 提醒：code=0，扣积分
