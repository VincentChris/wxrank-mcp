# 实时获取公众号推文列表

##### 简要描述

- 实时获取公众号推文列表（支持分页）

##### 收费标准

- `按调用次数：5分钱/次`

##### 请求URL

- ` http://data.wxrank.com/weixin/getps`

##### 请求方式

- GET / POST

##### 参数

| 参数名 | 必选 | 类型   | 说明                                                   |
| :----- | :--- | :----- | ------------------------------------------------------ |
| key    | 是   | string | 密钥                                                   |
| wxid   | 是   | string | 微信号或原始ID（gh开头）                               |
| cursor | 否   | string | 分页游标，用来获取下一页。`注意：cursor有效期是24小时` |

> `注意：wxid尽量采用原始ID（一般以gh或wxid开头），微信号是可以被修改的，原始ID无法修改不会变`
> `注意：每页可以获取10次推文，每次推文最多是8篇文章，通过art_url字段中idx可以判断位于第几篇？`

##### 请求示例

```
curl --location 'http://data.wxrank.com/weixin/getps' \
--header 'Content-Type: application/json' \
--data '{
    "key": "xxx",
    "wxid": "gh_e036770fc439",
    "cursor": "ZlM4Yk9jWlRXNnBwZE5iTGl6Nm43cUtiMVpCQ2daM2VtRi90amJmTzYyQT0="
}'
```

##### 返回参数说明

| 参数名        | 类型   | 说明                                             |
| :------------ | :----- | ------------------------------------------------ |
| list.pub_time | string | 文章发布时间                                     |
| list.art_url  | string | 文章发布链接                                     |
| list.pic_url  | string | 文章配图链接                                     |
| list.title    | string | 文章标题                                         |
| list.sn       | string | 文章唯一值，不会重复                             |
| wxid          | string | 公众号原始ID，一般以gh开头                       |
| cursor        | string | 分页游标（首页传空，传当前页cursor来获取下一页） |

> 注意：文章列表是没有总条数，当list数组等于[]的时候，说明已经是最后一页

##### 备注

> `code 状态判断：`

- 0：获取成功
- 1000：积分不足
- 1001：wxid不能为空
- 1002：wxid验证失败
- 1003：cursor校验失败
- 1004：请求失败，请重试
- 1005：请求超时，请重试
- 9999：QPS超过上限，请稍后再试（每秒不能超过5个）

> 提醒：code=0，扣积分
