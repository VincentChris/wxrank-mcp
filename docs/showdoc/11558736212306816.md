# 实时获取公众号推文列表（短链）

##### 简要描述

- 实时获取公众号推文列表（返回短链）

##### 收费标准

- `按调用次数：5分钱/次`

##### 请求URL

- ` http://data.wxrank.com/weixin/getpc`

##### 请求方式

- GET / POST

##### 参数

| 参数名 | 必选 | 类型   | 说明                                       |
| :----- | :--- | :----- | ------------------------------------------ |
| key    | 是   | string | 密钥                                       |
| biz    | 是   | string | 公众号biz，例如：MzI2NDk5NzA0Mw==          |
| begin  | 否   | int    | 开始位置，默认0；每页返回5次推文，以此类推 |

##### 请求示例

```
curl --location 'http://data.wxrank.com/weixin/getpc' \
--header 'Content-Type: application/json' \
--data '{
    "key": "xxx",
    "biz": "MzI2NDk5NzA0Mw==",
    "begin": 0
}'
```

##### 返回参数说明

| 参数名                                         | 类型   | 说明                                              |
| :--------------------------------------------- | :----- | ------------------------------------------------- |
| begin                                          | int    | 分页开始位置，首页是0                             |
| total_count                                    | int    | 总推文次数，不是文章总数。一次会出现推送多篇文章  |
| publish_list                                   | list   | 推送文章列表（每页返回5次的推文，按时间顺序返回） |
| publish_list.msgid                             | int    | 推送号，对于当前公众号是唯一的                    |
| publish_list.sent_total                        | int    | 送达人数，可以间接预估粉丝量                      |
| publish_list.sent_info_time                    | string | 推送时间                                          |
| publish_list.sent_appmsg_list.title            | string | 文章标题                                          |
| publish_list.sent_appmsg_list.art_url          | string | 文章发布链接                                      |
| publish_list.sent_appmsg_list.pic_url          | string | 文章配图链接                                      |
| publish_list.sent_appmsg_list.idx              | int    | 文章位置，1.头条，以此类推                        |
| publish_list.sent_appmsg_list.is_deleted       | int    | 是否删除，0.否 1.是                               |
| publish_list.sent_appmsg_list.copyright_status | int    | 版权状态：11.原创，101/201.转载                   |

##### 备注

> `code 状态判断：`

- 0：获取成功
- 1000：积分不足
- 1001：系统繁忙，请稍后再试
- 9999：QPS超过上限，请稍后再试（每秒不能超过2个）

> 提醒：code=0，扣积分
