# 实时获取公众号搜索列表

##### 简要描述

- 实时获取公众号搜索列表（支持分页）

##### 收费标准

- `按调用次数：1毛钱/次`

##### 请求URL

- ` http://data.wxrank.com/weixin/getsu`

##### 请求方式

- GET / POST

##### 参数

| 参数名  | 必选 | 类型   | 说明            |
| :------ | :--- | :----- | --------------- |
| key     | 是   | string | 密钥            |
| keyword | 是   | string | 搜索关键词      |
| page    | 否   | int    | 分页页码，默认1 |

##### 请求示例

```
curl --location 'http://data.wxrank.com/weixin/getsu' \
--header 'Content-Type: application/json' \
--data '{
    "key": "xxx",
    "keyword": "丁香医生"
}'
```

##### 备注

> `code 状态判断`

- 0：获取成功
- 1000：积分不足
- 1001：关键词不能为空
- 1002：请求失败，请重试
- 1003：请求超时，请重试
- 1008：请求频繁，请稍候
- 9999：QPS超过上限，请稍后再试（每秒不能超过3个）

> 提醒：code=0，扣积分
