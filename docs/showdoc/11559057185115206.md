# 实时获取文章内容页（含阅读数）

##### 简要描述

- 实时获取公众号文章页内容（包括阅读、点赞、分享数等）

##### 收费标准

- `按调用次数：5分钱/次`

##### 请求URL

- ` http://data.wxrank.com/weixin/artdata`

##### 请求方式

- POST

##### 参数

| 参数名 | 必选 | 类型   | 说明                                                                                      |
| :----- | :--- | :----- | ----------------------------------------------------------------------------------------- |
| key    | 是   | string | 密钥                                                                                      |
| url    | 是   | string | 文章链接，仅支持长链（例如：https://mp.weixin.qq.com/s?__biz=xxx==&mid=xxx&idx=1&sn=xxx） |

##### 请求示例

```
curl --location 'http://data.wxrank.com/weixin/artdata' \
--header 'Content-Type: application/json' \
--data '{
    "key": "xxx",
    "url": "https://mp.weixin.qq.com/s?__biz=MzIzOTU0NTQ0MA==&mid=2247544558&idx=1&sn=fb57861a2cbb1e24d5f297d303725362&chksm=e8e822f97fd977ad2b1b484c72bdb2024bbe6305aef91735edf67e471670d714cae5a4a352ff&scene=126&sessionid=1736469183#rd"
}'
```

> python语言调用示例

```
import requests
import json

apikey = 'your_api_key';
url = 'http://data.wxrank.com/weixin/artdata'
data = {
    "key": apikey,
    "url": "https://mp.weixin.qq.com/s?__biz=MzIzOTU0NTQ0MA==&mid=2247544558&idx=1&sn=fb57861a2cbb1e24d5f297d303725362&chksm=e8e822f97fd977ad2b1b484c72bdb2024bbe6305aef91735edf67e471670d714cae5a4a352ff&scene=126&sessionid=1736469183#rd"
}
headers = {
    'Content-Type': 'application/json'
}
response = requests.post(url, headers=headers, data=json.dumps(data))
print(response.text)

```

##### 返回参数说明

| 参数名           | 类型   | 说明                                                 |
| :--------------- | :----- | ---------------------------------------------------- |
| biz              | string | biz                                                  |
| mid              | int    | mid                                                  |
| sn               | string | sn（文章唯一ID，不会重复）                           |
| idx              | int    | 文章位置，1代表头条，以此类推                        |
| article_url      | string | 文章长链接                                           |
| short_link       | string | 文章短链接                                           |
| name             | string | 公众号名称                                           |
| user_name        | string | 公众号原始id                                         |
| pub_time         | string | 文章发布时间                                         |
| signature        | string | 公众号简介                                           |
| hd_head_img      | string | 公众号头像                                           |
| msg_cdn_url      | string | 文章封面图                                           |
| service_type     | int    | 0.订阅号 1.服务号                                    |
| copyright_stat   | int    | 0.非原创 1.原创 2.转载                               |
| title            | string | 文章标题                                             |
| digest           | string | 文章摘要                                             |
| author           | string | 原创作者                                             |
| comment_id       | string | 文章留言id                                           |
| province_name    | string | 发布于（IP属地）                                     |
| msg_daily_idx    | string | 当天第几次推文，`有些号能一天推多次`                 |
| item_show_type   | string | 0:文章（最常见）5:纯视频 7:纯音乐 8:纯图片 10:纯文字 |
| picture_url_list | array  | 文章中的插图链接地址                                 |
| text             | string | 文章内容（去除html标签）                             |
| html             | string | 文章内容（包括html标签）                             |

> appmsg_bar_data （互动数据）

| 参数名               | 类型 | 说明                 |
| :------------------- | :--- | -------------------- |
| read_num             | int  | 阅读数               |
| like_num             | int  | 点赞数（大拇指）     |
| look_num             | int  | 在看数（爱心图标）   |
| share_num            | int  | 分享数（转发图标）   |
| collect_num          | int  | 收藏数（隐藏数据）   |
| reward_count         | int  | 赞赏数，又名喜欢作者 |
| comment_count        | int  | 留言数，也有叫评论数 |
| original_content_num | int  | 原创内容篇数         |

##### 备注

> `code 状态判断`

- 0：获取成功
- 1000：积分不足
- 1001：文章链接为空（必须是公众号文章长链接）
- 1002：文章验证失败（文章链接解析失败，验证不通过）
- 1003：系统出现故障（该情况稍后再试，可联系管理员）
- 1004：文章删除/遮蔽（这种情况表示文章已不能访问）
- 9999：QPS超过上限，请稍后再试（每秒不能超过5个）

> 提醒：code=0，扣积分
