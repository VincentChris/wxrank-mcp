# 实时获取公众号文章留言

##### 简要描述

- 实时获取公众号文章留言列表，支持分页（每页最多100条）
- 支持获取更多回复留言列表，看最下面有调用说明

##### 收费标准

- `按调用次数：5分钱/次`

##### 请求URL

- ` http://data.wxrank.com/weixin/getcm`

##### 请求方式

- GET / POST

##### 参数

| 参数名     | 必选 | 类型   | 说明                     |
| :--------- | :--- | :----- | ------------------------ |
| key        | 是   | string | 密钥                     |
| comment_id | 是   | bigint | 文章留言id               |
| buffer     | 否   | string | 分页游标，用来获取下一页 |

> 怎么获取comment_id呢？通过文章内容解析接口：https://www.showdoc.com.cn/2343746579263506/11558501783383049
> ![](https://www.showdoc.com.cn/server/api/attachment/visitFile?sign=698ad9a19d3b3834d83156c0643fe49e&file=file.png)

##### 请求示例

```
curl --location 'http://data.wxrank.com/weixin/getcm' \
--header 'Content-Type: application/json' \
--data '{
    "key": "xxx",
    "comment_id": "4182456389740265479",
    "buffer": ""
}'
```

> python语言调用示例

```
import requests
import json

apikey = 'your_api_key';
url = 'http://data.wxrank.com/weixin/getcm'
data = {
    "key": apikey,
    "comment_id": "4182456389740265479",
    "buffer": ""
}
headers = {
    'Content-Type': 'application/json'
}
response = requests.post(url, headers=headers, data=json.dumps(data))
print(response.text)

```

##### 返回参数说明

| 参数名                                   | 类型   | 说明                                                         |
| :--------------------------------------- | :----- | ------------------------------------------------------------ |
| buffer                                   | string | 分页游标                                                     |
| total_count                              | int    | 留言总数                                                     |
| comment_total_cnt                        | int    | 一级留言总数（不包括回复的），可以用此参数来判断是否需要翻页 |
| comment_list                             | list   | 留言列表，包括一级留言和回复留言                             |
| comment_list[].id                        | int    | 一级留言ID，唯一值                                           |
| comment_list[].content                   | string | 留言内容                                                     |
| comment_list[].like_num                  | int    | 留言点赞数                                                   |
| comment_list[].create_time               | string | 留言时间                                                     |
| comment_list[].ip_wording                | string | IP属地信息                                                   |
| comment_list[].nick_name                 | string | 留言人昵称                                                   |
| comment_list[].logo_url                  | string | 留言人头像                                                   |
| comment_list[].reply_new                 | string | 回复留言，如果没有回复内容，该字段不存在                     |
| comment_list[].reply_new.reply_total_cnt | int    | 回复留言总数                                                 |
| comment_list[].reply_new.reply_list      | list   | 回复留言列表                                                 |

#### 回复留言（如果有需要回复留言列表，可以这样调用。用offset分页，每页也是最多100条）

##### 请求参数

| 参数名       | 必选 | 类型   | 说明                                       |
| :----------- | :--- | :----- | ------------------------------------------ |
| key          | 是   | string | 密钥                                       |
| comment_id   | 是   | bigint | 文章留言id                                 |
| content_id   | 是   | bigint | 一级留言中的id                             |
| max_reply_id | 是   | bigint | 一级留言中的reply_new.max_reply_id         |
| offset       | 否   | int    | 偏移量，首页默认是0，第二页传100。以此类推 |

```
curl --location 'http://data.wxrank.com/weixin/getcm' \

    "content_id": "4308035335293501536",
    "max_reply_id": "20"
}'
```

> `code 状态判断`

- 0：获取成功
- 1000：积分不足
- 1002：获取失败，请重试
- 9999：QPS超过上限，请稍后再试（每秒不能超过3个）

> 提醒：code=0，扣积分
