# 实时获取公众号文章阅读

##### 简要描述

- 实时获取公众号文章阅读、点赞、在看数

##### 收费标准

- `按调用次数：2分钱/次`

##### 请求URL

- ` http://data.wxrank.com/weixin/getrk`

```
// 尽量采用post请求方式，减少url编码出现的问题
curl --location 'http://data.wxrank.com/weixin/getrk' \
--header 'Content-Type: application/json' \
--data '{
    "key": "xxx",
    "url": "https://mp.weixin.qq.com/s?__biz=MzUzMjY0NDY4Ng==&mid=2247504367&idx=1&sn=09d0f7e717a0bd73f7b515fe222270a9#rd",
    "comment_id": "4232577569528594434"
}'
```

> python语言调用示例

```
import requests
import json

apikey = 'your_api_key';
url = 'http://data.wxrank.com/weixin/getrk'
data = {
    "key": apikey,
    "url": "https://mp.weixin.qq.com/s?__biz=MzUzMjY0NDY4Ng==&mid=2247504367&idx=1&sn=09d0f7e717a0bd73f7b515fe222270a9#rd",
    "comment_id": "4232577569528594434"
}
headers = {
    'Content-Type': 'application/json'
}
response = requests.post(url, headers=headers, data=json.dumps(data))
print(response.text)

```

##### 请求方式

- GET / POST

`如果以get方式请求，url务必要做urlencode，因为url链接中往往会有&连接符，需要转义处理`

##### 参数

| 参数名     | 必选 | 类型   | 说明                                                                                        |
| :--------- | :--- | :----- | ------------------------------------------------------------------------------------------- |
| key        | 是   | string | 密钥                                                                                        |
| url        | 是   | string | 文章链接，仅支持长链接（例如：https://mp.weixin.qq.com/s?__biz=xxx==&mid=xxx&idx=1&sn=xxx） |
| comment_id | 否   | string | 文章留言id，如果需要返回留言数，需要提供该参数                                              |

> url若是短链，需要提前转一下长链，通过文章解析接口：https://www.showdoc.com.cn/2343746579263506/11558501783383049
> ![](https://www.showdoc.com.cn/server/api/attachment/visitFile?sign=5b9e336184fa040a1a34c6fff3868ae1&file=file.png)

##### 返回示例

```
{
    "code": 0,
    "data": {
        "read_num": 100001,
        "like_num": 4246,
        "look_num": 977,
        "share_num": 4635,
        "collect_num": 1130,
        "reward_count": 70,
        "comment_count": 17
    },
    "msg": "剩余875积分"
}
```

##### 返回参数说明

| 参数名        | 类型 | 说明                         |
| :------------ | :--- | ---------------------------- |
| read_num      | int  | 阅读数                       |
| like_num      | int  | 点赞数（大拇指）             |
| look_num      | int  | 在看数（爱心图标）           |
| share_num     | int  | 分享数（转发图标）           |
| collect_num   | int  | 收藏数（隐藏数据）           |
| reward_count  | int  | 赞赏数，又名喜欢作者         |
| comment_count | int  | 留言数，传comment_id才会返回 |

##### 备注

> `code 状态判断`

- 0：获取成功
- 1000：积分不足
- 1001：文章链接为空（必须是公众号文章链接）
- 1002：文章验证失败（文章链接解析失败）
- 1003：获取失败，请重试
- 1004：服务异常，请重试
- 9999：QPS超过上限，请稍后再试（每秒不能超过10个）

> 提醒：code=0，扣积分
